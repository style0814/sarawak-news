# Static / Compliance Pages

## Overview

Static pages for legal compliance (AdSense, PDPA) and site information.

## Pages

### About (`/about`)

**File:** `app/about/page.tsx`

- Mission statement and site description
- Feature highlights
- Coverage areas (Sarawak divisions)
- News sources list
- Contact information
- Multi-language support (EN/ZH/MS)

### Privacy Policy (`/privacy`)

**File:** `app/privacy/page.tsx`

- Data collection practices
- Cookie usage
- Third-party services (AdSense, analytics)
- User rights (PDPA Malaysia compliance)
- Data retention policy

### Terms of Service (`/terms`)

**File:** `app/terms/page.tsx`

- Usage terms and conditions
- Content disclaimer (news aggregator, not original publisher)
- User responsibilities
- Intellectual property

### Pricing (`/pricing`)

**File:** `app/pricing/page.tsx`

- Feature comparison card showing all features are FREE (RM 0/forever)
- Lists all available features with checkmarks
- Links to donate page for voluntary support
